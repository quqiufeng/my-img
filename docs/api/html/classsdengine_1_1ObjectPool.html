Detailed Description

template<typename T>
class sdengine::ObjectPool< T >

通用线程安全对象池模板

Template Parameters
T池化对象类型（必须是完整类型）

通过复用已分配的对象实例，减少 new/delete 开销。 支持自定义创建器和重置器，以及最大池容量限制。
